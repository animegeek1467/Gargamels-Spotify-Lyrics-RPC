# Gargamels-Spotify-Lyrics-RPC
 Gargamel's Spotify Lyrics RPC v1.0  Live Spotify synced lyrics for Discord Rich Presence.  ## Setup  1. Download the ZIP. 2. Extract it. 3. Open Discord. 4. Play something on Spotify. 5. Double-click `START_HERE.bat`. 6. Pick Easy Mode. 7. Log into Spotify. 8. Done.  If Easy Mode does not work, use Hard Mode in the setup wizard.
